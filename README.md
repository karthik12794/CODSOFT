# CODSOFT
UI/UX Design Internship Tasks

This repository contains three UI/UX design tasks completed as part of my internship practice using Figma. The goal of these tasks was to practice designing simple, clean, and user-friendly interfaces while focusing on layout, readability, and user experience.

Task 1 – Signup Flow
In this task, I designed a basic signup flow that allows users to create an account easily. The interface focuses on simplicity and smooth navigation so that users can complete the signup process without confusion.

Task 2 – Email Template
This task involved creating a responsive email template for newsletters or promotional messages. The design includes a header section, key information blocks, and clear call-to-action buttons to make the email visually appealing and easy to read.

Task 3 – Restaurant Menu UI
In this task, I designed a restaurant menu interface where users can browse food items, view details, and proceed to checkout. The design focuses on organized categories, clear product information, and a simple ordering flow.

These tasks helped me understand basic UI/UX design principles such as layout structure, visual hierarchy, and creating interfaces that are both functional and visually engaging.
